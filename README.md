# Chambre des Clés

## Overview

*Chambre des Clés* is a lightweight web application for managing short‑term rental reservations. It provides an administrative dashboard, client and reservation listings, and the ability to generate PDF invoices for each reservation.

The project is built with plain PHP, uses **Dompdf** for PDF generation, and includes a minimal Docker environment for easy deployment.

---

## Features

- Dashboard showing key statistics (current reservations, upcoming stays, etc.)
- CRUD‑style listings for clients and reservations
- Automatic PDF invoice generation with a clean, responsive template
- Secure session handling with `SameSite=Strict`, `HttpOnly` and `Secure` flags
- Composer‑based dependency management isolated to the `vendor/` directory
- Minimal Docker image based on `php:8.2-apache` with only required extensions

---

## Requirements

- PHP >= 8.0
- Composer (for dependency installation)
- Docker (optional, for containerised deployment)
- MySQL or compatible MariaDB database

---

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ChambreDesCles.git
   cd ChambreDesCles
   ```

2. **Install PHP dependencies**
   ```bash
   composer install
   ```
   This will download Dompdf and its required libraries into the `vendor/` directory.

3. **Configure the database**
   - Create a MySQL database (e.g., `chambre_des_cles`).
   - Update the credentials in `app/utils/db.php`.

4. **Set up the web server**
   - If you are using Apache, ensure the `DocumentRoot` points to the `app/` directory.
   - Enable `mod_rewrite` to allow the `.htaccess` routing.

---

## Usage

- Access the application via `http://localhost/` (or the appropriate host).
- The dashboard (`app/index.php`) displays an overview of reservations.
- Navigate to **Clients** and **Reservations** pages to manage data.
- From a reservation detail page, click **"Imprimer la facture (PDF)"** to generate and view the invoice.

---

## Docker Deployment (optional)

A minimal Docker configuration is provided.

1. **Build the image**
   ```bash
   docker compose build
   ```

2. **Run the container**
   ```bash
   docker compose up -d
   ```

   The container maps the `app/` directory to `/var/www/html` and mounts the `vendor/` directory so that Composer‑installed libraries are available.

---

## Generating PDF Invoices

PDF invoices are generated by `app/generatePDF.php` using a configurable HTML template. The template pulls reservation data from the database and includes branding, client details, service breakdown, and totals.

The logo, brand name, and other textual elements can be customized at the top of `generatePDF.php` through a set of `$cfg_…` variables.
